# 급여 관리 프로그램

---

#### 1. 프로젝트 목적

- 특성화된 자격증 및 경력 수당관리로 효율적인 급여관리시스템 구축
- 정상적인 급여관리를 통한 정량평가화



#### 2. 프로젝트 설명

- 자격증수당과 경력수당지급의 시스템화
- 새로운 지식습득장려와 성취동기 제공
- 직무수행능력 개선



#### 3. 프로젝트 기대효과

- 사원개인의 시장가치 향상 효과
- 이직과 결근 감소효과
- 장기적이고 안정적인 인력확보 효과
